Syntax Guide
Normal Identifiers: [A-Za-z_][A-Za-z_0-9]* {{ example }}
aVariable (variable names typically start with lower case letters)
another_variable (underscores are allowed except at the start)
ClassName (class names typically start with upper case letters)
Quoted Identifiers: '...'
Enumeration Definition: type X = enumeration(...); {{ example }}
type AssertionLevel = enumeration(warning, error);
type StateSelect = enumeration(never, avoid, default, prefer, always);
Vector Literals: {..., ...}
Array Literals: [..., ...; ..., ...]
Array Comprehension: { f(i) for i in 1:ni } {{ example }}
Two indices { f(i,j) for i in 1:ni, for j in 1:nj }
