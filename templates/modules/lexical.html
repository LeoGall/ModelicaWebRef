Normal Identifier Examples

Quoted Identifiers
Syntax: '...'

If an identifier is surrounded by single quotes, it can contains spaces, digits and non-digits anywhere in the name. Furthermore, it can contain any of the following characters:

! # $ % & ( ) * + , - . / : ; < > = ? @ [ ] ^ { } | ~
